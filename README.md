## Notice

在调用API之前需要在Redis中设置trojan-panel:jwk-key和trojan-panel:token:${username}

| key                              | 说明      |
|----------------------------------|---------|
| `trojan-panel:jwt-key`           | jwt的key |
| `trojan-panel:token:${username}` | 认证      |
